# Local Text Analyzer

## 소개
Streamlit 기반의 로컬 텍스트 분석 토이 프로젝트입니다.
외부 API 없이 텍스트의 기본 통계, 키워드, 간단 요약, 감정 분류를 제공합니다.

## 실행 방법
1. 가상환경 활성화
2. 패키지 설치
   pip install -r requirements.txt
3. 실행
   python -m streamlit run app.py

## 주요 기능
- 글자 수 / 단어 수 / 문장 수 분석
- 상위 키워드 추출
- 간단 요약
- 규칙 기반 감정 분류

# 지원 정보
- python 3.10
- streamlit 1.55.0
